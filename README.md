# Cистема для приёма и управления заявками на устранение недочетов в строительной компании

# Технологический стек:
- React с TypeScript
- Mobx
- React Router v6
- Material-UI (MUI)
- Axios
- Mokky dev

# Функционал
- Менеджер:
- вкладка "Объекты": CRUD строительных объектов (фильтрация, сортировка, пагинация);
- вкладка "Заявки": Управление заявками (изменение статуса, фильтрация, сортировка, пагинация);
- вкладка "Аналитика": Дашборд с аналитикой (диаграмма по статусам, графики по объектам и трендам по времени).

- Пользователь (без авторизации):
- форма подачи заявки с автоматическим расчётом приоритета (1–10) на основе критичности, категории, длины описания, файлов, времени подачи, истории объекта;
- страница отслеживания заявки по уникальной ссылке (статус, история, файлы).
